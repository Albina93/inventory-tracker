# Typescript and Object Oriented Programming

Typescript enforces type safety by ensuring that all variables, properties and method parameters follow defined types at compile time.
Inheritance reduces code duplication by allowing subclasses to reuse properties and methods from the base Product class.
PhysicalProduct and DigitalProduct inherit sku, name, price and shared methods.
Encapsulation and access modifiers help protect data, control how it's used, and create a cleaner, more maintainable code structure.
